MacOS Keyboard Navigation

Share
2022 Feb 26 PST - Revision as of: 2024 Feb 14 PST

Ok, so I feel like a complete idiot right now.

I've been trying to use Tab to navigate webpages on my MacBook for what feels like forever now. Turns out, it was as simple as hitting Ctrl+F7.

To be honest, I thought that it was just an issue with FireFox, and then I read Browser keyboard navigation in macOS by Scott Vinkle for The A11Y Project and realized that it was an issue with MacOS! This should really be enabled by default, but Apple is stupid.

Just thought I'd share this encase anyone else was having this issue.
